Some things I'm proud of

Oh Noah You Didn't

An online multiplayer game created for the sheridan game-making contest. This game won the award for "best use of code".

A Bit Off

An educational game created with Java Swing for ICS12U final culminating project. It teaches the basic bitwise operators.

Circuit Diagram Quizzer

A C# windows form application that generated various circuit diagrams and will test the user to enter their own values.

VEX Robotics Competitions

Participated and won various challenges/competitions using VEX kits and RobotC coding language.

All-In-One Music Aid

All in one device: musical tuner, metronome, and drone. Python GUI, Arduino, 3D printed enclosure.

Arduino LED Sequencer

An Arduino project that will read a sequence of data off an SD card, then output pattern to a strip of WS2811 LEDs.
